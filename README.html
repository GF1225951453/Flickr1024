Flickr1024: Dataset for Stereo Image Super-resolution

Yingqian Wang   Longguang Wang   Jungang Yang   Wei An   Yulan Guo

###
***Flickr1024 is a large stereo dataset, which consists of 1024 high-quality images pairs and covers diverse senarios. This dataset can be employed for the task of stereo image super-resolution.***
